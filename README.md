### Flask-React Starter Project

```bash
# Install Backend dependencies
pip install -r requirements.txt
```

```bash
# Install Frontend dependencies
npm install [--save | --save-dev]
```

```bash
# watch Frontend
npm run watch-js
```

```bash
# build Frontend
npm run build
```

```bash
# run all tests
pytest
```

```bash
# run app
py run.py
```
