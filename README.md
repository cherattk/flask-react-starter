### Flask-React Starter Project

##### Install Backend dependencies
```bash
pip install -r requirements.txt
```

##### Install Frontend dependencies
```bash
npm install [--save | --save-dev]
```

##### watch Frontend
```bash
npm run watch-js
```

##### build Frontend
```bash
npm run build
```

##### run all pyton tests
```bash
pytest
```

##### run app
```bash
py run.py
```
